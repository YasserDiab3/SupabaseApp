# تشغيل الترحيل والنشر من PowerShell

## 1. الانتقال إلى مجلد Backend (المسار فيه مسافة — استخدم علامتي الاقتباس)

```powershell
cd "d:\App\v.2-ok run\SupabaseApp\Backend"
```

أو إذا كنت داخل المجلد الرئيسي للمشروع:

```powershell
cd SupabaseApp\Backend
```

## 2. تشغيل سكربت النشر (ترحيل + نشر الدالة)

```powershell
.\deploy.ps1
```

إذا كان المشروع مرتبطاً مسبقاً وتريد تخطي الربط:

```powershell
.\deploy.ps1 -SkipLink
```

## 3. تشغيل الترحيل فقط (بدون نشر الدالة)

```powershell
supabase db push
```

---

**ملاحظة:** إذا كنت داخل `Backend\supabase` وليس داخل `Backend`، فالسكربت موجود في المجلد الأعلى:

```powershell
cd ..
.\deploy.ps1
```
